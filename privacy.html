Privacy Policy – Shift-Tap

Shift-Tap respects the privacy of its users. This policy explains what data is collected and how it is used.

Account Information

Users sign in using Google authentication. Basic account information such as email address and user ID may be used to identify the account.

Stored Data

Work hours, recovery days and user settings are stored securely in Google Firebase and linked to the user account.

Data Sharing

Shift-Tap does not sell or share personal data with third parties. Firebase is used only as a technical storage provider.

Security

All data is transmitted using secure HTTPS connections.

Data Deletion

Users can request deletion of their account and associated data by contacting:

shifttap.app@gmail.com

Account Deletion

Users can request deletion of their account and all associated data by sending an email to:

shifttap.app@gmail.com

When a deletion request is received, the following data will be removed:

  • User account identifier
  • Stored work hours
  • Recovery days
  • User settings

Deletion requests are processed within 30 days.
